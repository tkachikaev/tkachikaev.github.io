# timur.dev

Статический сайт для GitHub Pages.

## Публикация
Загрузить содержимое этой папки в корень репозитория `tkachikaev.github.io`.
В Settings → Pages выбрать ветку `main`, папку `/(root)`.

## Что менять
- Текст — `index.html`
- Цвет акцента — переменная `--accent` в `css/style.css`
- Картинки — папка `assets/gallery`
