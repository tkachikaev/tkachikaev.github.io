# Личная страница Timur

Статический сайт для GitHub Pages. Никакой сборки и Node.js не нужно.

## Структура

```text
index.html
css/style.css
js/main.js
assets/
  avatar.png
  favicon.svg
  gallery/
```

## Как опубликовать

1. Распакуй содержимое этой папки в репозиторий `tkachikaev.github.io`.
2. В GitHub нажми **Add file → Upload files** и перетащи все файлы и папки.
3. Сделай commit в ветку `main`.
4. В **Settings → Pages** убедись, что выбран `Deploy from a branch`, ветка `main` и папка `/(root)`.
5. Через пару минут открой `https://tkachikaev.github.io`.

## Что менять в первую очередь

- Тексты — `index.html`.
- Внешний вид, отступы, цвета и анимации — `css/style.css`.
- Скролл-анимации, мобильное меню и окно картинок — `js/main.js`.
- Фото — `assets/avatar.png`.
- Игровые картинки — `assets/gallery/`.

Фото профиля и визуальные карточки хранятся локально, поэтому сайт не зависит от внешних ссылок.
